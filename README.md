# Professional Portfolio Website

A modern, responsive single-page website built with HTML, CSS, and JavaScript featuring smooth animations and professional design.

## Features

### 🎨 Design & Layout
- **Responsive Navigation Bar** - Mobile-friendly hamburger menu
- **Hero Section** - Background image with animated text and call-to-action buttons
- **About Section** - Profile image with detailed description and contact information
- **Skills Section** - Animated progress bars showing technical proficiency
- **Projects Gallery** - Interactive portfolio with hover effects and project details
- **Testimonials** - Client reviews with star ratings
- **Contact Form** - Validated form with success/error notifications
- **Footer** - Social media links and site information
- **Scroll to Top Button** - Smooth navigation back to top

### ⚡ Interactive Features
- Smooth scrolling navigation
- Animated skill progress bars
- Hover effects on project cards
- Form validation with user feedback
- Mobile-responsive hamburger menu
- Parallax scrolling effects
- Intersection Observer animations
- Active navigation link highlighting

### 📱 Responsive Design
- Fully responsive layout for all devices
- Mobile-first approach
- Tablet and desktop optimizations
- Touch-friendly interface

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **JavaScript (ES6+)** - Interactive functionality
- **Font Awesome** - Icon library

## Getting Started

1. **Download the files** to your local machine
2. **Open `index.html`** in your web browser
3. **No installation required** - runs entirely in the browser

## Customization

### Personal Information
Update the following in `index.html`:
- Name and title in the header
- About section content
- Contact information
- Project details
- Testimonials

### Styling
Modify `styles.css` to:
- Change color schemes
- Adjust fonts and typography
- Update animations
- Customize responsive breakpoints

### Functionality
Edit `script.js` to:
- Add new animations
- Modify form behavior
- Update scroll effects
- Add interactive features

## Mobile Responsiveness

### ✅ **Fully Responsive on All Mobile Devices**
